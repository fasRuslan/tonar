Полуприцеп сортиментовоз Тонар L3

Полуприцеп-сортиментовоз для перевозки длинномерных грузов длиной 2х6м или 3х4м (лесные материалы, трубы и.т.д.).

Варианты исполнения коников:
- разборные / телескопические.

Технические характеристики

Комплектации

  • Модель 9445
  • ССУ 1150 / 1250 / 1350
  • Тип полуприцеп-сортиментовоз для перевозки длинномерных грузов
  • Кузов коники разборные / телескопические
  • Объем геометрический объём перевозимого груза: 63 / 90 куб.м
  • Кол-во осей 3
  • Тип ошиновки односкатный
